## 快速上手

### 通过 NPM 安装和运行

1. 进入项目目录：
   ```bash
   cd your-project
   ```

2. 安装依赖项：
   ```bash
   npm install
   ```

3. 启动开发服务器：
   ```bash
   npm run electron:dev
   ```

### 通过 YARN 安装和运行

1. 进入项目目录：
   ```bash
   cd your-project
   ```

2. 安装依赖项：
   ```bash
   yarn
   ```

3. 启动开发服务器：
   ```bash
   yarn run electron:dev
   ```

## 注意事项

- 确保你已安装 Node.js 和 NPM/YARN。
